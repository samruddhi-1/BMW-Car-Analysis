Project Overview
This project analyzes BMW car data to identify trends in pricing, mileage, fuel type, and transmission. The goal is to provide insights for business decisions like inventory management and pricing strategy.

Dataset

Source: Kaggle
Key columns: Model, Year, Price, Mileage, Fuel Type, Transmission


Tools & Technologies

Python (Pandas, NumPy, Matplotlib, Seaborn)
Jupyter Notebook
Power BI for dashboards


Analysis Performed

Price distribution across BMW models
Impact of year and mileage on price
Comparison of fuel types and transmission
Identification of premium vs budget models
