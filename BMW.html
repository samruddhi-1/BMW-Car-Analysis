1. Project Planning

1.1 Problem Defination

The goal of this project is to analyze Car prices and understand what factors affect the price of a car. For Example, we will check if the model, year, mileage,fuel type, or engine size has an impact on the price. This can help buyers or sellers estimate a fair price for used cars.

1.2 Business Objectives

  • To find which features (like mileage,engine size , or year) have the strongest effect on car price.
  • To Understand how fuel type or transmission affects pricing
  • To build a simple model that can predict the price of a car based on its details.

1.3 Project Scope

  • Dataset: Used BMW Car Dataset
  • Tools: Python, Jupyter Notebook , Power BI
  • Libraries: pandas , numpy, matplotlib , seaborn (for analysis and visualization)
  • Timeline:
    • Data understanding and cleaning
    • Exploratory Data Analysis(EDA)
    • Visualization and Insights
  • Expected Output: Cleaned data,meaningful charts , and a short summary of findings

def remove_outliers(df):
    #df = df[
        #(df['year'] >= 2012) & (df['year'] <= 2019) &
        #(df['price'] >= 4000) & (df['price'] <= 35000) &
        #(df['mileage'] >= 5000) & (df['mileage'] <= 70000) &
        #(df['tax'] >= 20) & (df['tax'] <= 180) &
        #(df['mpg'] >= 30) & (df['mpg'] <= 50) &
        #(df['engineSize'] >= 1.0 )& (df['engineSize'] <= 2.5)
   # ]
    for col in['year','price','mileage','tax','mpg','engineSize']:
            lower_q = 0.05
            upper_q = 0.95

            if col == 'year':
                lower_q,upper_q = 0.02,0.98
            elif col == 'tax':
                lower_q,upper_q = 0.20,0.98
            elif col == 'engineSize':
                lower_q , upper_q = 0.15,0.85
            elif col == 'mpg':
                lower_q, upper_q = 0.10,0.90
        
            q1 = df[col].quantile(lower_q)
            q9 = df[col].quantile(upper_q)
            df = df[(df[col] >= q1) & (df[col] <= q9)]
            
        
    return df.reset_index(drop = True)

df1 = remove_outliers(df1)
